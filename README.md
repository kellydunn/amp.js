# what

A javascript audio histogram for your phat beats or smoof jamz.

# why

I like watching bouncy basslines in my browser.

# support

HTML5 ain't easy.
Chrome and Firefox only.

```

Amp.Manager.url = "/your/mega/nasty/track.mp3"
```
